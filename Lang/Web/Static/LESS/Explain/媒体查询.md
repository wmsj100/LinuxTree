媒体查询使用`@media`来实现；

```css
.screencolor{
    @media screen {
        color: green;
        @media (min-width: 768px) {
            color: red;
        }
    }
}
body{
    .screencolor;
}
```

当页面的宽度大于`768px`时候，页面字体为`red`，其余字体颜色为`green`；
