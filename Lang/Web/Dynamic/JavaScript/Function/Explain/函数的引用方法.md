title: 函数的引用方法
date: 2016-06-18

函数也是对象,所以引用函数就像引用对象,其实也只是引用的堆栈中的一个指针,所以就像对象一样,函数也可以有多个引用值.
函数可以被当作参数调用,也可以当作值被返回,如下:

```javascript
    function fn1(fn2, args) {
        return fn2(args);
    }

    function fn3(num) {
        return num + 10;
    }
    var a = fn1(fn3, 20);   //30;
```
