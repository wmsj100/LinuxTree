- 重复与回溯

```javascript
    var str = "<p>Para 1.</p>" +
              "<img src='smiley.jpg'>" +
              "<p>Para 2.</p>" +
              "<div>Div.</div>";
    str.match(/<p>.*<\/p>/i);   //贪婪模式
    //["<p>Para 1.</p><img src='smiley.jpg'><p>Para 2.</p>"]
    str.match(/<p>.*?<\/p>/i);  //非贪婪模式
    //["<p>Para 1.</p>"]
```

- 贪婪模式 -- 先匹配尽可能多的内容，一般都是直接到底，然后再回溯，知道符合条件之后停止。
- 非贪婪模式 -- 先匹配尽可能少的内容，如果匹配，就停止，否则就增加一个字符然后再进行验证，知道符合条件就停止。

`.` -- 代表一个字符
`.*` -- 匹配零个或多个
`.*?` -- 匹配零个或多个但尽可能少重复，
