# common 比较文本

- 逐行比较已经排序的俩个文件。显示结果包括3列：第一列：只在第一个文件中找到的行；第二列：只在第二个文件中找到的行；第三列：俩个文件共有的行。
- 该命令逐行比较已经拍过序的俩个文件；
- 和jon和uniq一样，只能处理已经拍过序的文件

## 参数
- -1 不输出文件1特有的行
- -2 不输出文件2特有的行
- -3 不输出俩个文件共有的行

## 实例
- comm f1.txt f2.txt  
- comm -12 f1.txt f2.txt  值显示俩个文件共有的内容
