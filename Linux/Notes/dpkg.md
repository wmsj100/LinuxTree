# dpkg

## 概要

- dpkg是debian系统的一个命令工具，用来安装/删除/构建软件包

## 用法

- `sudo dpkg -i mysql` 安装软件包
- `sudo dpkg -r mysql` 卸载软件
- `sudo dpkg -l` 查看已经安装的软件
- `sudo dpkg --help` 查看帮助
