# dmidecode

## 概要

- Linux上面用来查看系统硬件信息的工具,

## 执行

- `sudo dmidecode` 打印所有硬件信息
- `sudo dmidecode -t bios` 只打印bios相关的信息
- `sudo dmidecode -t memory` 查看内存信息
```
