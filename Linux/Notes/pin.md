# pin

## 概要

- Personal identification Number 就是SIM卡的个人识别密码，
- PIN码是一种保护SIM卡的安全措施，防止别人盗用SIM卡
- 三次输入错误PIN码后手机便会自动锁卡并提示输入PUK码，再SIM卡背面
- 输入10次错误的PUK码手机卡自动报废
