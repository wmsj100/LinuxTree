- linux/window/mac 换行符转换
- unix2dos file 把linux换行符转换为windows的换行符
- dos2unix file 把windows的换行符转换为unix的
- unix2mac file 把unix的换行符转换为mac的换行符
- mac2unix file 把mac的换行符转换为unix的换行符
