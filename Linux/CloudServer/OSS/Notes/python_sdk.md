# oss python sdk

## 概要

- oss为python提供的sdk工具包

## 安装

- `pip install oss2`
- `import oss2; oss2.__version__;` 查看oss2的版本
- `import crcmod._crcfunext` 验证功能是否完好
- `pip uninstall oss2` 卸载
