# 创建文件/文件夹

- mkdir -p father/son/grandson 这样可以创建一个多层级的目录结构。这在安装软件和配置安装路径时非常有用。

- mkdir -v test 显示创建新目录的信息
- mkdir -vp shiyanlou/{lib/,bin/,doc/{info,product}} 一次创建多个同级目录并且包含内容。
