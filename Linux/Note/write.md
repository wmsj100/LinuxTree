# write

## 概述
- 在Linux中这个命令可以用来对tty窗口进行通信

## 用法
- write pi pts/1  这样就可以进行通信了
- 前提是只能在一台服务器上的`tty`终端进行通信，不能通过远程IP方式进行通信
