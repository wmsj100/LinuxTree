# 安全组

## 概述
- 安全组通常是公有云或者私有云服务的一个微服务，用于端口放行，公有云默认是禁止所有端口接入虚拟机的，需要手动添加要放行的端口，
- 通常只需要设置入方向添加端口就可以

## 场景
- 我在阿里云的虚拟机上安装了mariadb后无法通过pymysql连接数据库，刚开始以为是防火墙禁用了，后来意识到是安全组没有放行，添加之后就可以连接了。
