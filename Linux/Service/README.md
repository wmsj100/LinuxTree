# README

## 概要

- 云服务厂商的服务
- 现在肯定不会自己购买服务器了，肯定是使用云服务厂商提供的服务，最常用的比如ECS云服务器，OSS对象存储
- 每个厂商提供的服务功能类似，但名称可能不同。比如华为的OBS，在阿里云就是OSS
- 虽然服务的名称和参数配置可能不同，但整体原理是类似的

## 服务

- OSS: 阿里云的对象存储服务
