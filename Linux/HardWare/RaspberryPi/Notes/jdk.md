# 安装jdk

## 概要

- 配置java环境，然后就可以启动java服务

## 步骤

- `sudo apt search jdk` 选择Oracle的java8版本
- `sudo apt install oracle-java8-jdk` 安装好java环境
- `java --version` 查看java版本
- `javac -version` 查看javac版本
