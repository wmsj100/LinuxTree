# iproute2

## 概要

- 之前对于网络的管理都使用工具net-tool `ifconfig/route`等分割的数个工具,但那是过时的而且工具本身已经不再维护
- 现在通过iproute2工具包来管理,这是现在新发行版默认配置的网络工具
