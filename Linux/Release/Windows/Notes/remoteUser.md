# remoteUser

## 概要

- windows开启远端连接需要配置如下
- 点击“计算机”属性，开启远端连接，取消复选框勾选
- 关闭计算机防火墙
- 点击防火墙的高级选项，进入安全组弹框
- 点击入站规则，查找Remote User 3389启用，3个RemoteUser 都启动
- mstsc 连接
