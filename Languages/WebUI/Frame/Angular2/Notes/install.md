# install

## 概要

- 安装angular环境

## 操作

- `cnpm install -g @angular/cli`
- `cd ~/workspace/angular`
- `ng new my-app` angular会自动下载,但是下载的源是在国外,等文件创建完成手动下载就好
- `cd my-app && cnpm install` 用淘宝镜像下载
- `ng server --open --host 0.0.0.0 --port 4200` 启动服务
