# 声明文件

## 概述
- 在`ts`中使用第三方库,需要使用声明,不然ts并不知道库指的是什么;
- `declare var jQuery:(string) => any;`
- 约定声明文件以`*.d.ts`;
- 第三方声明文件推荐使用工具`@types`统一管理;
- `npm install @types/jquery --save-dev`;
