# ncu

## 概要

- 是一个检测和升级package.json的软件到最新版本的工具

## 使用

- npm install -g npm-check-updates
- ncu 检测当前目录的package.json
- ncu -u 更新全部软件到最新版
- npm install 安装最新版本
