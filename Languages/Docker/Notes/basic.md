## 安装

- `sudo apt install docker.io` 这样安装的版本较低
- `wget -qO- https://get.docker.com/ |sh` 这样安装的是官方的最新版本，建议按照这个安装，而且这样安装的是社区版本。

- `sudo usermod -aG docker pi` 把当前用户添加到docker组
