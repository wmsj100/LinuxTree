# commit

## 概要

- 这个命令可以把当前的容器导出到images中，
- 只有把容器提交成为镜像，然后才可以通过对镜像操作来实现容器的转移
- `docker commit -a wmsj100 -m "gcc-9.2.0" centos7_wmsj100`
