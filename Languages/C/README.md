## 方向

- Linux 开发/游戏后台开发
- 需要在 C/C++ 的基础上再学习 Linux 操作系统，主要包括 Linux 基本操作、Shell、文件系统、进程线程、内存、Socket 通信、内核等，甚至还需要与算法、Qt 等相结合。
- 另外，也可以使用Go语言进行 Linux 开发，Go语言在全球已经有相当多的应用案例了。
- 游戏的后台服务器大部分也是基于 Linux的，也会用到以上技能。
