# os_stat

## 概要

- 用于在给定路径上执行一个系统stat调用

## 返回值

- `st_uid`: 所有者用户ID
- `st_gid`: 组ID
- `st_atime`: 上次访问时间
- `st_mtime`: 最后一次修改时间
- `st_ctime`: 最新的元数据更改时间
