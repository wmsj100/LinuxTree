# encode和decode

## 概要

- encode: 用于将str类型转换为bytes类型，对原字符串进行编码，不会直接修改原字符串，如果想修改原字符串，需要重新赋值。
- decode: 将bytes类型的二进制数据转换为str类型，这个过程也称为解码。
