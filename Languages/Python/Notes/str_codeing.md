# 字符编码

## 概要

- python的字符编码是`utf-8`
- `sys.getdefaultencoding()` `utf-8` 获取当前环境的编码格式
- `ord('Q')` `81` 输出当前字符对应的编码数字
- `chr(81)` `Q`  输出当前数字对应的编码`
