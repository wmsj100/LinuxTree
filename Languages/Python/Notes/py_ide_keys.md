# python自带IDE快捷键

## 概要

- 写python可以使用自带的IDE来调试

## 热键

- `alt + p` 上一条历史命令
- `alt + n` 下一条历史命令
- `alt + /` 自动补全代码，可以连续按来切换
- `alt + 3` 注释代码
- `alt + 4` 取消注释代码
- `alt + g` 跳行
- `alt + z` 撤销
- `alt + shift + z` 恢复
- `alt + ]` 缩进代码
- `alt + [` 取消缩进代码
