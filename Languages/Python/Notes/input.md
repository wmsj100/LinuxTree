# input

## 概要

- input是用来获取用户输入的，但实际上这个函数应该只是在测试时候有价值，因为真正不可能通过它来获取用户输入。
- 所有用户的输入都是通过界面来接收的，因为界面可以进行数据的校验，然后通过接口传值给python进行处理。
- input获取用户的输入，并且把所有的输入都转换为字符串，这和通过界面传达给接口的值类似，都是以字符串形式下发。
