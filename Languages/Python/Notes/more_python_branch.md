# more_python_branch

## 概要

- python的版本管控可以通过pipenv来切换，但前提是当前系统有按照对应的python版本
- python版本可以通过编译安装，
- 按照多个python版本后通过`pipenv --python 3.6`来分别实例化对应的python环境，这样不会互相影响
