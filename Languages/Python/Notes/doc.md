# doc

## 概要

- 函数的说明文档

## 使用

- 通常在函数的内部顶部书写
- 用单引号包裹

## 范例

```python
def hello():
	'''
	比较俩个值的大小
	'''
	pass

hello.__doc__ #'\n\t提供函数的比较大小\n\t'
```
