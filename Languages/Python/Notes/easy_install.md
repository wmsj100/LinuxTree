# easy_install

## 概要

- `easy_install`是python的一个包管理器，类似pip，如果安装pip会自动安装这个包
- 该工具用来安装和管理`egg`包
- `easy_install-3.6 dmPython-2.3-py3.6-linux-x86_64.egg`
- 对于包的卸载可以借助于pip来实现
