# sys模块

## 概述

- 该模块提供对解释器使用或维护的一些变量的访问，以及与解释器强烈交互的函数。

## 常用方法

- `sys.argc`: 传递给python脚本的命令行参数列表
    - sys.argv[0] == 文件名
	- sys.argv[1] 是文件名后面的值
