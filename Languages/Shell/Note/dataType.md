# 数据类型

## 概述
- shell中每一个变量的值都是字符串，无论给变量赋值时候有没有使用引号，都会以字符串的形式存储
- shell默认情况下不区分变量类型，即使赋值整数或浮点数，都被视为字符串
- 赋值号的周围不能有空格
