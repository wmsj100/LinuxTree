闭包就是为了读取函数内部的局部变量或者是为了保存一个`for`循环过程中的`i`的值，而使用闭包的方式有一下几种，

- 使用一个自执行函数进行读取，在自执行函数内部创建局部变量来保持i的值。

```javascript
for(var i=0;i<4;i++){
    (function(){
        var num = i;
        arr[num].onclick = function(){
            console.log(num);
        }
        })();
}
```

当作参数传入

```javascript
for(var i=0;i<4;i++){
    (function(num){
        arr[num].onclick = function(){
            console.log(num);
        }
        })(i);
}
```
