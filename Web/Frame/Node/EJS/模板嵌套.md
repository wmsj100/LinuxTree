`ejs`模板可以层层嵌套，如下配置

目录文件

- demo4.ejs
- menu
    + menu_item.ejs
    + menu
        * itme.ejs

路由文件

```javascript
router.get('/ejs/demo4', function(req, res){
    res.render('demo4');
}); 
```

demo4.ejs
```html
    <ul>
        <% var url = "foo" %>
        <% var title = "foo" %>
        <% include menu/menu_item %>

        <% var url = "bar" %>
        <% var title = "bar" %>
        <% include menu/menu_item %>

        <% var url = "baz" %>
        <% var title = "baz" %>
        <% include menu/menu_item %>
    </ul>
```

menu/menu_item.ejs
```html
<li><% include menu/item %></li>
```

menu/menu/item.ejs
```html
<a href="/<%= url %>" title="<%= title %>"><%= title %></a>
```

这个文件嵌套有三层，
模板嵌套也是使用`js`格式
`<% include menu/item %>` 这样就可以了。
