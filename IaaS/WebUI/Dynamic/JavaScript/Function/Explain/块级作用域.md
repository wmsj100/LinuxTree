---
title: 块级作用域
date: 2016-06-22
tags: [Function]
categories: Dynamic
---

JS是没有块级作用域的，但是可以通过`闭包`来模拟。

- 函数内部的块级作用域

```javascript
    function fn(){
        (function(){
            var i = 0;
        })();
        console.log(i); //error
    }
```

- 全局函数的块级作用域

```javascript
(function fn(){
        var num = 1;
    })();
console.log(num);   //error
```

在匿名函数中定义的任何变量，在执行完毕之后都会被销毁。

这种方法可以避免污染全局环境

这种方法可以减少闭包占用内部问题，因为没有指向匿名函数的引用，只要函数执行完毕，就可以立即销毁其作用域链了。
