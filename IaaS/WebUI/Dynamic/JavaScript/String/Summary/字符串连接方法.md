常见的字符串的连接方法有一下几种：

- 通过`+`连接符 -- **推荐**--效率最高的方法。

```javascript
var a = "";
a = a + "b" + "asdf";
a;  //"basdf";
```

- 通过`+=`；这个通过用于`HTML`内容的增加。

```javascript
var a = "";
a += "b";
a += "asdf";
a;  //"basdf"
```

- 通过`join`方法连接数组；

- 通过`concat`方法连接字符串，`concat`不是仅仅用来连接数组的，字符串也是可以的。

```javascript
var a="asdf";
a.concat("wmsj","qwe");
"asdfwmsjqwe"
var b=[1,2,3];
b.concat(4,5)
[1, 2, 3, 4, 5]
```
