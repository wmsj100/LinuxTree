## 动机
- 想要知道当前的目录结构，
- 在一个文件中备份当前目录的所有文件结构，方便以后查阅，
- 文件的所有变化一目了然

## 软件介绍
- tree ./ 查看当前目录内的文件结构

```
./
├── finger.md
├── tree.md
└── unix2dos.md

0 directories, 3 files
```
