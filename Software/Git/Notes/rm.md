# rm

## 概要

- 从git库中删除文件

## 操作

- `git rm flask/study4/logs/error.log` 产生`删除文件
- `git rm -f flask/study4/logs/error.log` 当文件在本地有修改时删除会告警，可以添加f参数强制删除
