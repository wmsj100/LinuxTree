# 多行缩进

## 概述
- 选择`v`进入可视模式
- 按下`Shift >`可以向右缩进
- `Shift <` 向左缩进

- `Shift V`进入多行选择的可视模式也可以进行上面的操作
- `.` 通过按下`.`来重复执行上次操作
