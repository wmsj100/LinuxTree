# tab

## 概要

- vim7版本之后支持了多标签功能

## 使用

- `:tabnew file` 在新标签打开目标文件
- `:tabclose ` 关闭当前tab
- `:tabonly ` 关闭所有打开的tab，只保留当前的tab
- `:tabprevious` 前一个tab
- `:tabNext` 后一个tab
- `gt` 在tab之间切换
