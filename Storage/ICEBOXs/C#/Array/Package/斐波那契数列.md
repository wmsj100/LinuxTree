date: 2016-08-15
tags: [C#]

```c#
static int Fn2(int n)
        {
            if (n == 1 || n == 2)
            {
                return 1;
            }
            else
            {
                return Fn2(n - 1) + Fn2(n - 2);
            }
        }
```
