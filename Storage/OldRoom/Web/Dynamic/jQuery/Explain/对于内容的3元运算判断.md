---
title: 对于内容的3元运算判断
date: 2016-07-04
tags: [jQuery]
categories: Dynamic
---

场景是我想要通过判断链接的title值是否为空，来决定是否要创建一个`span`元素，而我的判断依据就是一个3元运算

```javascript
var link = $("a");
var span = link[0].title ? <span>link[0].title : "";
var $span = $(span);
$("div").append($span);
```

如果链接的`title`值为空，那么span就是一个空字符，而`append`一个空字符是不会有任何影响的。
