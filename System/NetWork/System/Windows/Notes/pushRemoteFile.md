# 远程服务器传输文件

## 远程共享文件夹
- 在远程服务器上创建一个文件夹`soft`
- 共享`soft`给`everyone`，并且赋予读取和写入权限
- `ipconfig` 获取到远程服务器主机IP地址
- 在本地文档浏览器地址栏输入`\\服务器IP`
- 在弹出的用户窗口输入服务器用户名和密码，用户名需要带域
