# RaspberryPi

## 概述
- 全球最流行的微型计算机，可以实现很多操作，当前最新型号是`3b+`，
- 可以用来当作代码托管/个人博客/服务器场景操作，还可以接入一些外设实现自动化智能化操作
- 当前了它的外设听说性价比比较低
- 2019-04-17 购买树莓派一台
