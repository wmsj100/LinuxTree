# win10打开系统设置

## 概述
- win10的系统设置打开方式和win7不同,没有计算机图标,右击我的电脑属性也没有

## 操作
- 点击`文件管理器`左上角的`对号`图标就可以进入;
- 右击`文档管理器`的`此电脑`图标,选择`属性`也可以进入;
