# 删除windows.old

## 概述
- 系统升级时候会在c盘自动生成一个`windows.old`文件夹,大小会在十几个G左右,占用大量磁盘空间,该目录只是用于系统还原时候使用,正常情况没有用;
- 右击`我的电脑`选择`属性`
- 点击`清理磁盘`
- 清理完成后在`磁盘清理`对话框找到`清理系统文件`点击
- 程序扫描完成后在`要删除的文件`中勾选`以前的Windows安装`
- 点击确定重新开始清理;
