# Hadoop

## 概要

- hadoop是大数据的处理,曾经辉煌过,是大数据的标准,现在再和云计算争霸时候落败,
- hadoop是将计算和存储结合在一起,而现在主流是计算和存储分离.
- 现在hadoop的计算会被spark替代,存储会被obs对象存储替代
