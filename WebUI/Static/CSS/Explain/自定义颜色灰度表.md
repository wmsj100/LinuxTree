date: 2016-06-16

```html
<style>
    .p1{
    color: rgb(0,0,0);
}
    .p2{
        color: rgb(20%,20%,20%);
    }
    .p3{
        color: rgb(40%,40%,40%);
    }
    .p4{
        color: rgb(60%,60%,60%);
    }
    .p5{
        color: rgb(80%,80%,80%);
    }
</style>
<p class="p1">hello world</p>
<p class="p2">hello world</p>
<p class="p3">hello world</p>
<p class="p4">hello world</p>
<p class="p5">hello world</p>
```
