## 代办
- `RxJS` `reactive extensions library for javascript` ==> RxJS是一个使用Observables进行反应式编程的库，可以更容易地组合异步或基于回调的代码。这个项目是Reactive-Extensions / RxJS的重写，具有更好的性能，更好的模块化，更好的可调试调用堆栈，同时保持大部分向后兼容，有一些突破性的变化，减少了API表面
