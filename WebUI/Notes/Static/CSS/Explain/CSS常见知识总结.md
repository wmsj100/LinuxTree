背景由内容+内边距+边框组成

内边距和边框主要的作用是装饰，外边距主要用于布局，

`relative`可以设置偏移量和`z-index`;
绝对定位的块级元素在宽度未定义的时候，不再是默认的`100%`了，而是根据内容自动撑开。而且会默认覆盖其他元素

`+`相邻元素选择器，的使用有俩个条件：
 - 二者必须拥有同一个父元素；
 - 二者相邻

 `属性选择器` -- IE9之前的浏览器无法兼容
 ```css
 [title]{
     color: red;    /* 所有拥有title属性的元素的文字颜色设置为红色 */
 }
 a[href][title]{
     color: red;    /* 同时拥有href和title属性的a标签文字 */
 }
 ```
