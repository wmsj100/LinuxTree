# linux基础知识

- linux里面一切皆文件。

- 一个目录需要同时有读权限和执行权限才额可以打开并查看内部文件。

- adduser 和 useradd的区别在于，后者只是创建用户，创建完成后需要使用passwd去设置用户秘密，而adduser会创建用户/目录/秘密，更像是执行一种程序。
