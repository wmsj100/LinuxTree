# 提交

## 概述
- git commit -m "commit name"

## 修改已经提交的内容
- git reset --hard commitid 硬撤销，完全恢复，
- git reset --soft commitid 软恢复
 
## 修改已提交注释
- git commit --amend 可以进入提交信息页面修改具体提交内容
