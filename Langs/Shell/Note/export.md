# export

## 概述
- export命令将作为它参数的变量导出到子shell中，并在子shell中生效
- 在默认情况下，一个shell中被创建的变量在这个shell调用的下级(子)shell中是不可使用的。
- export命令把自己的参数创建为一个环境变量，而这个环境变量可以被当前程序调用的其它脚本可见和使用
