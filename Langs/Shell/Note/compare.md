# 比较运算

## 概述
- `eq` 相等
- `ne` 不相等
- `gt` 大于
- `lt` 小于
- `ge` 大于等于
- `le` 小于等于

## 使用
- 通常在if条件句中使用
```sh
if [ $a -eq $b ]
then
	echo "a is equal b"
fi
```
