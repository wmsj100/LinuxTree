# 运算符

## 概述
- 关系运算符的结果是布尔值
- 0: 为假
- !0: 为真，真值通常为1

## 优先级
- 赋值《 逻辑运算符《 关系符《 运算符《 非

## 类别
- 一目运算符: ++/--
- 二目运算符 +/-/*///%/+=/-=/*=//=/%=
- 三目运算符 a>b?a:b 条件为真返回a，否则返回b
