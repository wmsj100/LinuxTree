# NULL

## 概述
- 对于未初始化的变量，都赋值为NULL，
- NULL通常是指向内存0的地址，该地址是操作系统访问权限，程序没有访问权限
- `int *p=NULL`

## void
- void表示指针指向的数据类型未知
