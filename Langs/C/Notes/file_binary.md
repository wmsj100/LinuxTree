# 二进制文件

## 概述
- 文件分为俩类，二进制文件和文本文件。
- 从物理上讲二进制和字符文件没有区别，都是以二进制的形式存储，当他们在文件的组织形式上不一样
- 二进制文件有文件头，用以表明文件的大小/类型等信息，程序处理二进制文件时候会先分析文件头，判断文件是否合法。
- 文件头后面的数据才是程序真正要处理的；
- 字符文件没有文件头，第一个字节就是要显示的内容。
