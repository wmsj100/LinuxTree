利用递归的方法实现：

```c#
static int Fn(int n)
    {
        if (n <= 1)
        {
            return 1;
        }
        else
        {
            return n * Fn(n - 1);
        }
    }
...
int num = 5, sum;
sum = Fn(num);
Console.WriteLine(sum);
```
