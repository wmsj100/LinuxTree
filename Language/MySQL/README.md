# MySQL

## 概述
- 关系型数据库，通过二维表来定位，不能存储俩条完全相同的数据，必须有一个值来区分数据
- 通用的语言为SQL
- 数据库作为专用语言，规则很少，所以很容易学习，肯定没有学习一门语言花费时间
