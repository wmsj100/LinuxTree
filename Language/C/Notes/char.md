# 字符

## 概述
- 只能表示ASCII字符
- 使用''表示单个字符，长度位1
- 使用`putchar`或者`printf`打印字符

## 存储
- 计算机存储字符时存储的不是字符本身，而是字符在字符集中的编号/编码值。
- 字符和整数类型本质上没有什么区别
- 可以给字符类型赋值一个整数，或者以整数的形式输出字符。
- 也可以给整数类型赋值一个字符，或者以字符的形式输出整数类型。
